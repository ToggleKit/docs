Slider

A slider displays one image at a time, allowing visitors to navigate through images using next and previous buttons. Each transition is accompanied by a smooth animation. Multiple sliders can be implemented on a single page, each functioning independently. Additionally, these sliders can be customized with various transition effects and navigation styles to enhance user experience.

Structure

<div data-rt="sdc" data-tg="sliderBox">
  <div data-rt="sdr" data-tg="slideable">
    <img src="https://picsum.photos/id/237/300/500" alt=""/>
    <img src="https://picsum.photos/id/227/300/500" alt=""/>
    <img src="https://picsum.photos/id/235/300/500" alt=""/>
    <img src="https://picsum.photos/id/230/300/500" alt=""/>
    <img src="https://picsum.photos/id/238/300/500" alt=""/>
    <img src="https://picsum.photos/id/225/300/500" alt=""/>
    <img src="https://picsum.photos/id/234/300/500" alt=""/>
    <img src="https://picsum.photos/id/231/300/500" alt=""/>
  </div>
  <button data-rt="prev" data-tg="prev">Prev</button>
  <button data-rt="next" data-tg="next">Next</button>
</div>
