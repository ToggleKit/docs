Content Pass

Content pass is make a copy of code and sent to the tergeted tag as child . This component receive 3 parameters 1. data-container for parent element 2. data-aim='' for terget element where the content going tp pass 3. data-fire for sending data by clicking

Structure

<div data-container data-aim="#terget">
<ol data-code>
<li>List 1</li>
<li>List 2</li>
<li>List 3</li>
<li>List 4</li>
<button data-fire>Pass</button>
</ol>
</div>
<div id="terget"></div>
