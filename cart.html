Cart

Users can add their favorite items to a cart. The selected items will remain in the cart until the user decides to remove them. Each page can have only one cart, but the cart can contain multiple items. Additionally, users can easily manage their cart by adding or removing items as they wish, ensuring a seamless selecting experience.

Selectors

 [data-tg="cart"] = cart div where all selected items will be visible
 [data-tg="selectedItem"] = selected items in cart
 [data-tg="itemRemoveBtn"] = item remove button in cart
 [data-tg="cItem"] = main item container
 #id = post/article id [unique]
 [data-tg="cURL"] = item post/article anchor tag
 [data-tg="cImg"] = post/article thumbnail image
 [data-tg="cTitle"] = post/article main title
 [data-tg="cAdd"] = checkbox for selection favourite items

Note: Users can add their favorite items to a cart, allowing them to explore and review these items later. The selected items will remain in the cart until the user decides to remove them.
