Modal

A modal is a popup window that appears on top of the main content of a webpage. It contains text content and provides a boolean choice, typically in the form of “Yes” or “No” buttons. Visitors can read the text content within the modal and make a selection by clicking one of the buttons. This interaction often requires the visitor to make a decision or provide confirmation before they can proceed. Multiple modals can be used on the same page, each serving different purposes, such as displaying alerts, collecting user input, or providing additional information without navigating away from the current page.

Selectors

[data-tg="eventer"] = show modal container by clicking this buttons
[data-tg="shadow"] = shadow of the modal (customisable)
[data-tg="container"] = modal main container (customisable)
[data-tg="header"] = header of the modal
[data-tg="header"]>h3 = modal title
[data-tg="closer"] = modal closer button
[data-tg="body"] = modal body
[data-tg="body"]>p = modal text content (customisable)
[data-tg="footer"] = modal footer
[data-tg="saver"] = visitor can agree with condition by clicking this button
[data-tg="canceller"] = visitor can disagree with condition by clicking this button
