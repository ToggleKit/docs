Nest Menu

The nestMenu component serves as the navigator for a website, converting a list of links into a dropdown navigation menu. Note that only one nestMenu can be added to a webpage. This component also adds class names to the dropdowns, allowing for custom styling. Additionally, it enables you to specify the container for the dropdown navigation bar and define the breakpoints.

Structure

<div data-tg='nestMenu'>
  <ul>
    <li><a href="#"> Services </a></li>
    <li><a href="#">_service 1</a></li>
    <li><a href="#">__service 1.1</a></li>
    <li><a href="#">Projects</a></li>
    <li><a href="#">_Project 1</a></li>
    <li><a href="#">_Project 2</a></li>
    <li><a href="#">__Project 2.1</a></li>
    <li><a href="#">__Project 2.2</a></li>
  </ul>
</div>

Convert into


<div data-tg="nestMenu">
  <ul>
    <li class="on-sub">
      <a href="javascript:void(0)"> Services </a>
      <ul class="sub-menu">
        <li class="drop-1">
          <a href="javascript:void(0)">service 1</a>
          <ul class="sub-sub-menu">
            <li class="drop-2">
              <a href="#">service 1.1</a>
            </li>
          </ul>
        </li>
      </ul>
    </li>
  
    <li class="on-sub">
      <a href="javascript:void(0)">Projects</a>
      <ul class="sub-menu">
        <li>
          <a href="#">Project 1</a>
        </li>
        <li class="drop-1">
          <a href="javascript:void(0)">Project 2</a>
          <ul class="sub-sub-menu">
            <li class="drop-2">
              <a href="#">Project 2.1</a>
            </li>
            <li class="drop-2">
              <a href="#">Project 2.2</a>
            </li>
          </ul>
        </li>
      </ul>
    </li>
  </ul>
</div>

Selectors


[data-tg='nestMenu'] = main container
.on-sub = parent menu
.sub-menu = first sub menu container (ul)
.sub-menu.show = add class name after .on-sub clicked
.drop-1 = first sub menu lists (li)
.sub-sub-menu = second sub menu container (ul)
.sub-sub-menu.show = add class name after .drop-1 clicked
.drop-2 = second sub menu lists (li)
