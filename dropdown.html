Drop Down

A drop-down is a list of items that appears after clicking a button. On a single page, there can be multiple drop-downs, and each drop-down can interact with the others. This functionality allows for a more dynamic and interactive user experience. For instance, selecting an option in one drop-down can filter or change the options available in another. This interconnected behavior is particularly useful in forms and complex interfaces, making navigation and data entry more efficient and user-friendly.

Structure

<div data-tg="dropDown">
  <button>Drop Down</button>
  <div>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
    <li>Item 4</li>
    <li>Item 5</li>
  </div>
</div>
