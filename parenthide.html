Parent Hide

Parent Hide is a function that hides the parent element when its child button is clicked. This can be particularly useful for creating dynamic and interactive web pages. For example, you might use this function to remove a notification or a specific section of a form when the user clicks a close button.

Structure

<div data-tg="parentHide">
  <button>hide</button>
  <p>Lorem ipsum dolor sit amet, consectetur adipisicing elit.</p>
</div>
