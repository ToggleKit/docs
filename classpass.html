Class Pass

Class Pass components accepts two parameters:

  • 1. data-aim='#target' : Specifies the element to which the class will be added.
  • 2. data-class='active' : Specifies the class name to be toggled.

    • When the host element is clicked, the specified class will be toggled on the target element. The data-pass='' attribute can contain any valid CSS selector.

    Structure
